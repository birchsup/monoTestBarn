  - Обновить/добавить проверки на некорректный формат id, отсутствие id и несуществующий id.
- **Критерии готовности (Definition of Done):**
  - Выбран и зафиксирован один канонический endpoint для получения test case.
  - Регистрация роутов и handlers соответствует выбранному контракту.
  - Интеграционные тесты отражают фактический контракт и проходят.
